# TLA Plus Foundation (tla-plus-foundation)
The TLA+ Foundation is a Linux Foundation project promoting the adoption of TLA+, a high-level specification language created by Leslie Lamport for modeling concurrent and distributed systems. With inaugural members AWS, Oracle, and Microsoft, it supports formal verification tools and education.
